    day2

    - tag.class : selects tag with this class only (higher specifity than tag selector)
    - tag1>tag2 : selects any tag2 child of tag1 (direct child selector)
    - tag1 tag2 : selects any tag2 inside of tag2 (direct or indirect)
    - tag1+tag2 : selects any tag2 direct adjacent of tag1
    - tag1~tag2 : selects any tag2 after tag1 inside the same parent (indirect adjacent)
    - tag1,tag2,tag3 : group selector (has same specifity as single selector)
    - tag[attribute=value]: attribute selector (exact equal)
    - tag[att^=val]: starts with
    - tag[att$=val]: ends with

    - < iframe> : src -> url (optional)
    - < iframe> : name -> to be used as a target for anchor tags

    - < table> : contains < tr> rows
    - < caption> : children of table
    - < tr> : contains < dt> cells
    - < dh> : replaces dt in headers

    - caption + thead + tbody + tfoot : to organize table parts

    - td and th attribute rowspan and colspan to expand cells
    - table attribute cellPadding and cellSpacing

    - input name is useed to add url arguments (query string) key-value pairs sep=&
    - label for to focus related input with the same id
    - input accesskey to create shortcut to focus on input (different shortcut keys different browsers) alt+shift (firefox)
    
    - input checkbox/raidobutton grouped by "name" and labeled with "value"

    - < select> for drop down
    - < optgroup> to add sections inside menu (label attribute)
    - < option> single item
    - size attribute for select to expand
    - multiple attribute to allow multiple selection
